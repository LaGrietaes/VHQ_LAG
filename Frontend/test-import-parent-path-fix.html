Parent Path Validation Tests

This test verifies that the import system correctly handles parent paths and prevents the "file already exists, mkdir" error.

Test Cases

  • Valid Parent Path: Should import files into the specified folder
  • Invalid Parent Path (File): Should import files to root instead of trying to create directory with file name
  • No Parent Path: Should import files to project root
  • Non-existent Parent Path: Should import files to root instead of failing
