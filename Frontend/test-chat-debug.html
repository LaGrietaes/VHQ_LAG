Chat Component Debug Test

Debug Information:

This page will help test the chat component functionality.

Open the browser console to see debug logs.

Test Steps:

  1. Open browser console (F12)
  2. Look for the chat button in bottom-right corner
  3. Click the chat button to open the drawer
  4. Select an agent using the checkboxes
  5. Type a message in the input field
  6. Press Enter or click the send button
  7. Check console for debug messages
